# Random Number Guessing Game

This Python program generates a random number between 1 and 10. The user keeps guessing until the correct number is entered.

## Features

- Random number generation
- Unlimited attempts
- User input
- Correct/Wrong messages

## Concepts Used

- random module
- randint()
- while loop
- if else
- input()
